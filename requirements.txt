# SYSTEM REQUIREMENTS NOTES:
# ===============================================
# 
# REQUIRED SYSTEM DEPENDENCIES:
# ------------------------------
# 1. FFmpeg (for video processing):
#    Ubuntu/Debian: sudo apt-get install ffmpeg
#    CentOS/RHEL:   sudo yum install ffmpeg
#    macOS:         brew install ffmpeg
#    Windows:       Download from https://ffmpeg.org/download.html
#
# 2. System fonts (for text rendering):
#    Ubuntu/Debian: sudo apt-get install fonts-dejavu-core
#    CentOS/RHEL:   sudo yum install dejavu-sans-fonts
#
# ENVIRONMENT VARIABLES REQUIRED:
# -------------------------------
# AZURE_SPEECH_KEY=your_azure_speech_key
# AZURE_SPEECH_REGION=your_azure_region
# SUPABASE_URL=https://your-project.supabase.co
# SUPABASE_KEY=your_supabase_anon_key
# GEMINI_API_KEY=your_gemini_api_key
# GROQ_API_KEY=your_groq_api_key
# PINECONE_API_KEY=your_pinecone_api_key
# MONGO_URI=your_mongodb_connection_string
# MONGO_DB_NAME=your_database_name
# COLLECTION_NAME=your_collection_name
#
# VIDEO GENERATION PIPELINE:
# --------------------------
# 1. Script Generation: Gemini API
# 2. Image Creation: PIL with educational templates
# 3. Audio Generation: Azure Cognitive Services TTS
# 4. Video Assembly: FFmpeg (system dependency)
# 5. Storage: Supabase cloud storage or local filesystem
#
# INSTALLATION COMMANDS:
# ---------------------
# pip install -r requirements.txt
# sudo apt-get install ffmpeg fonts-dejavu-core  # Ubuntu/Debian
# 
# DOCKER SETUP (if using containers):
# -----------------------------------
# FROM python:3.9
# RUN apt-get update && apt-get install -y ffmpeg fonts-dejavu-core
# COPY requirements.txt .
# RUN pip install -r requirements.txt
#
# PERFORMANCE NOTES:
# -----------------
# - FFmpeg provides much better performance than MoviePy
# - Parallel processing of video segments for maximum speed
# - Azure TTS provides high-quality audio generation
# - Supabase handles cloud storage automatically
# - Total video generation time: 3-5 minutes for 30-second educational videos
#
# TROUBLESHOOTING:
# ---------------
# 1. FFmpeg not found: Ensure FFmpeg is installed and in PATH
# 2. Font errors: Install system fonts (dejavu-sans-fonts)
# 3. Azure TTS errors: Check AZURE_SPEECH_KEY and AZURE_SPEECH_REGION
# 4. Supabase errors: Verify SUPABASE_URL and SUPABASE_KEY
# 5. Memory issues: Ensure sufficient RAM for parallel processing
#
